Focus Stacking & Manual Focus Peaking

Phase 3: Focus Like a Pro 🤖 Together Coder (Llama 3.3 70B) 200 XP

Objective

Master focus stacking for infinite depth of field in macro, and use focus peaking for razor-sharp manual focus.

What Is Focus Stacking?

Take multiple photos at different focus distances → combine in software → single image with infinite depth of field. Essential for macro, product, landscape foreground-to-background.

📋 Focus Stacking Workflow (R7)

1
Setup: Tripod mandatory. Manual focus. Manual exposure (same settings all shots). Remote shutter or 2-sec timer.
2
Focus Bracketing: Menu → Focus Bracketing → Enable. Set: Number of shots (10-50), Focus increment (1-10), Auto exposure bracketing OFF.
3
Set Start Point: Focus on nearest point you want sharp. Press shutter → camera shoots stack automatically.
4
Stack in Software: Canon DPP (free), Photoshop (File → Automate → Photomerge), Helicon Focus, Zerene Stacker.

R7 Focus Bracketing Settings

  • Number of shots: 10-20 (macro), 5-10 (landscape)
  • Focus increment: 1-3 (small steps = smoother), 5-10 (faster, risk gaps)
  • Auto exposure bracketing: OFF (keep exposure constant)
  • Flash: OFF (or consistent power if using)

🎯 Focus Peaking — Your MF Best Friend

Turn on: Menu → MF Peaking Settings → Peaking: ON • Color: Red/Yellow/Blue/White • Level: High/Medium/Low

  • What it does: Highlights in-focus edges with colored lines
  • Colors: Red (contrasty scenes), Yellow (general), Blue (low light), White (high contrast)
  • Level: High = more sensitive, Low = only sharpest edges
  • R7 bonus: Works in video mode too!

Manual Focus with Peaking Workflow

  1. Switch lens to MF (or body MF)
  2. Turn on Peaking (Red, High)
  3. Magnify: 5x or 10x (INFO button or touch 2x)
  4. Turn focus ring until peaking highlights appear on target
  5. Fine-tune: rock focus ring slightly back/forth to find peak
  6. Shoot

Stacking Software Comparison

  • Canon DPP (Free): Built-in, basic, OK for simple stacks
  • Photoshop: File → Automate → Photomerge → Collage, Auto blend layers
  • Helicon Focus: Best algorithms, depth map output, batch (paid)
  • Zerene Stacker: Best for macro, PMax/DMap methods (paid)
  • CombineZP: Free, older, learning curve
🎯 Challenge

Focus stacked macro + MF peaking demo. Shoot: (1) 15-shot macro stack of flower/insect (tripod, focus bracketing), (2) MF peaking: manual focus on 3 subjects at different distances using peaking. Compare stacked vs single shot.
