The Live Edit — Cull, Rate, Deliver

Phase 6: Event Photographer Mode 🤖 Together Coder (Llama 3.3 70B) 250 XP

Objective

Turn 1000 shots into 50 heroes in record time. In-camera culling, batch editing, and delivery workflow that clients love.

📋 The 5-Stage Culling Workflow

1
In-Camera (During Event): Rate 1-5 stars. 5=hero, 4=strong, 3=maybe, 2=backup, 1=delete. Do between shoots.
2
Import + Auto-Cull: Lightroom/DPP → Import 5★ + 4★ only. Flag 3★ for review. Reject 1-2★.
3
Rapid Cull (Additive Method): Grid view → P (pick) / X (reject). 3-5 seconds per image. Goal: 500 → 50.
4
Batch Edit: Sync settings across similar lighting. Exposure, WB, contrast, noise reduction. Preset per scene.
5
Export & Deliver: Web JPGs (2000px long edge, sRGB, 85% quality) + Full-res archive. Gallery link.

In-Camera Rating System (R7)

  • ★★★★★ (5): Portfolio hero, immediate hero
  • ★★★★ (4): Strong contender, needs minor edit
  • ★★★ (3): Maybe — review later
  • ★★ (2): Backup / group shot alternative
  • ★ (1): Delete (blink, blur, duplicate, bad)

Pro tip: Rate during downtime (speeches, dinner). Don't chimp every shot.

The Culling Mindset

  • Be ruthless: If you hesitate, it's not a 5
  • First instinct: Your gut knows in 2 seconds
  • Compare similar: Keep best of burst, delete rest
  • Technical first: Blink/blur → instant reject
  • Then artistic: Expression, composition, moment

Batch Editing Workflow

  1. Group by lighting scene (outdoor, indoor, flash, sunset)
  2. Edit ONE representative image per group
  3. Sync settings: Select all → Sync → Check all
  4. Fine-tune outliers individually
  5. Export preset: 2000px, sRGB, 85% quality, sharpen standard

Delivery That Wows

  • Gallery: Pixieset (free tier), SmugMug, Pic-Time, ShootProof
  • Password: Client name + event date
  • Download: Original + Web sizes
  • Mobile app: Client can favorite/download on phone
  • Expire: 1 year (or forever for weddings)
🎯 Challenge

Cull 50 → 10 → 5, edit 5. Import 50 RAW files (or use sample). Cull to 10 in 5 min. Cull to 5 in 3 min. Batch edit 5 with single preset. Export web + full. Time: < 30 min total.
